## Key Features to Highlight
- Dual C2 Modes: TCP/IP (Socket.IO) and Blockchain (Solana)
- Modern Electron GUI with real-time monitoring
- Advanced stealth features for forensic investigations
- Comprehensive device control (camera, mic, files, SMS, location, etc.)
- Chunked file transfer for large files
- Blockchain-based decentralized C2
- Cross-platform server (Windows, Linux, macOS)
- Android 5.0+ support
